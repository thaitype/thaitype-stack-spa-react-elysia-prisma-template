# Goal

Setup template like _ref/thaitype-stack-mongodb-template
but using 
- SPA App
- Elysia (https://elysiajs.com/) for the backend
- Prisma (https://www.prisma.io/) for the database layer
- React (https://react.dev/) for the frontend + Vite (https://vitejs.dev/) for the build system
- Tanstack Router
- Better Auth
- tailwindcss/shadcn/ui for the UI components
  
App 
- Todo App
- Minimal Clean Architecture, no need domain, just service/repository
